SolarNetwork Example: TypeScript Chart with billboard.js

This is an example app built with TypeScript that demonstrates how to use the SolarNetwork API with the help of the solarnetwork-api-core library to make authenticated API calls that drive the generation of an area chart using the billboard.js library. This is a good way for a front-end developer new to SolarNetwork to learn about using the API. Fill in the query details in this form and click the Generate button to generate a chart.

See the project on Github for more information.
